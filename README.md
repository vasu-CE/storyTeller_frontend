# Frontend Setup

## Prerequisites
- Node.js 18+
- Backend running on `http://localhost:3000`

## Install
```bash
cd frontend
npm install
```

## Environment (optional)
Create `frontend/.env`:
```env
VITE_API_URL=http://localhost:3000/api
```

## Run
```bash
cd frontend
npm run dev
```

App URL: `http://localhost:5173`

## Build
```bash
cd frontend
npm run build
```
